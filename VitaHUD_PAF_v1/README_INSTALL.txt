Install:
1. Upload/replace these files inside GitHub repo folder VitaHUD_PAF_v1.
2. Run GitHub Actions.
3. Download vitahud_paf_v5_suprx artifact.
4. Copy vitahud_paf_v5.suprx to ur0:tai/vitahud_paf_v5.suprx.
5. Ensure ur0:tai/config.txt has under *main:
   ur0:tai/vitahud_paf_v5.suprx
6. Full reboot.
7. Check ur0:data/VitaHUD/vitahud_alive.txt.
